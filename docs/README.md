# 知識庫文件目錄

歡迎使用組織知識庫。本目錄包含所有技術文件、FAQ 與操作指南。

## 目錄結構

```
docs/
├── getting-started/    # 入門指南
├── faq/               # 常見問題
└── technical/         # 技術參考文件
```

## 如何貢獻知識庫

1. 在對應分類目錄下建立 `.md` 文件
2. 使用下方的文件模板格式
3. `git push` 後系統會自動更新搜尋索引

## 文件模板格式

```markdown
---
title: 文件標題
tags: tag1, tag2
updated: 2026-01-01
---

# 文件標題

內容...
```

## 找不到答案？

請使用知識庫網站的搜尋功能，若找不到答案可以自動建立 Issue，AI 會自動嘗試回答。
